PA4-2 Report

Created2023-01-07|Updated2023-01-07|学习历程
|Post View:

一、题目大意

给出一张图$G$，$G$中有$n$个带权值的点，点$i$的权值为$t[i]$，点与点之间有$m$条连边，求从点$1$到点$n$之间的最短路长度并计算数目。
其中$1 \leq n,m \leq 100000$，$1 \leq t[i] \leq 10000$。

二、算法分析

本题是最短路计数模板题的变形，从边权值变形为点权值，但是实际实现时并无本质区别，因此可以采用狄杰斯特拉算法来解决。在实现过程中，对于点权值问题，我们从将要扩展的点的集合中取出一个点$u$进行扩展，此时对未进行扩展的点$v$，若$u$与$v$之间存在连边，则可以将后向点$v$的权值视作这条边的权值，如此处理就可以使用狄杰斯特拉算法进行计算。

算法正确性说明：在处理每条边时的顺序类似拓扑排序，每条边都被处理一次，因此可以用后向点权值代替整条边的权值。

三、代码实现

首先考虑图的存储问题。为了节省空间，采用链式前向星存储图的方式，定义edge结构体用来存储边的终点to和后继next，head数组中head[u]存储以u为起点的边，并定义addedge函数用来收集信息。（其中cnt和head数组的初始值均为$-1$）

实现了优先队列之后，我们就可以编写优化的狄杰斯特拉算法。首先定义优先队列pq，定义数组dis和sum分别用来存储最短路径长度和最短路径数目，定义数组pweight用来存储每个点的权值。算法开始时先将dis数组初始化为一个上界值inf=0x3f3f3f3f，之后令dis[1] = pweight[1]，sum[1] = 1作为结点1的初始值，进入优先队列。定义数组visited表示结点是否已被扩展，此后当优先队列不为空时，进行贪心更新操作：获取当前队列顶端点p，遍历与p相连的终点q，若dis[q] > dis[p] + pweight[q]，则更新dis[q]为dis[p] + pweight[q]，同时更新sum[q]为sum[p]，并将更新后的结点q入队。若上述两值相等，则更新sum[q] = sum[q] + sum[p]，注意题目要求取模即可。这样就实现了优化的狄杰斯特拉算法。核心代码如下：

四、实现中值得注意的细节

  1. 本题中边为双向边，因此在读取输入加边时，要同时进行addedge(u, v)和addedge(v, u)操作，保存边信息的数组大小为m * 2，否则会导致Wrong Answer。
  2. 由于运算过程中dis和sum数组中的结果可能过大，因此采用long long数据类型进行存储。

五、时间与空间复杂度分析

  1. 本题中，朴素狄杰斯特拉算法的时间复杂度为$O(n^2)$，优化后的狄杰斯特拉算法时间复杂度为$O(m\log{n})$。
  2. 本题中，用于存储边信息的空间为$O(m)$，存储点信息的优先队列的空间为$O(n)$，则整体的空间复杂度为$O(m+n)$。
